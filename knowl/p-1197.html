Note 2. If there is axial symmetry about z - axis, then \(u\) depens only upon \(r\) and \(\theta\) so that \(\phi =0 \) in equation (7.3.117),
\begin{equation} u(r,\theta) = \sum\limits_{n=0}^{\infty}\left(A_{n}r^{n}+\frac{B_{n}}{r^{n+1}}\right)\{C_{n}P_{n}(\cos\theta)+D_{n}Q_{n}(\cos\theta)\}\tag{7.3.118} \end{equation}
If \(D_{n}=0\text{,}\)
\begin{equation} u(r,\theta) = \sum\limits_{n=0}^{\infty}\left(A_{n}r^{n}+\frac{B_{n}}{r^{n+1}}\right)C_{n}P_{n}(\cos\theta)\tag{7.3.119} \end{equation}
