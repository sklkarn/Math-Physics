Eigenvectors help understand linear transformations easily. There are directions along which a linear transformation acts simply by stretching/compressing and/or flipping; eigenvalues give the factors by which this compression occurs. Consider a matrix \(A\) undergoing a physical transformation (e.g rotation). When this matrix is used to transform a given vector \(X\) the result is \(Y=AX\text{.}\) Now ask a question: Are there any vectors \(X\) which does not change their direction under this transformation, but allow the vector magnitude to vary by scalar \(\lambda\text{?}\) Such a question is of the form \(AX=\lambda X\) So, such special \(X\) are called eigenvector(s) and the change in magnitude depends on the eigenvalue \(\lambda\text{.}\) Eigenvalues characterize important aspect of linear transformations, such as whether a system of linear equations has a unique solution or not. In many applications eigenvalues also describe physical propereties of a mathematical model. In Quantum Mechanics, the eigenvectors of operators which corresponds to observable quantities, like energy, position, etc., form a complete basis for the space of all possible states of the system that youâ€™re analysing. That is, any state you want can be written as a linear combination of these eigenvectors. Naturally, this makes to solve problems much easier, all we need is to find the coefficients in this linear combination for which there is a neat formula.
